# kromo
`kromo` is a play on words, from "chromatic" and "lo-mo". I made `kromo` because perfect lenses are overrated.

## Usage
```
$ python3 kromo.py chosen8.jpg

Original Image details: JPEG (1962, 2615) RGB
Dimensions must be odd, automatically cropping...
New dimensions: (1961, 2615)
Completed in: 67.25719666481018s
```

## Runtime
`kromo` is slow, just like how film photography used to be.
